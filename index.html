  Important

  • Facebook accepts video in maximum 720p (1280 x 720) resolution, at 30 frames per second.
  • You must send an I-frame (keyframe) at least once every two seconds throughout the stream.
  • Recommended max bit rate is 2500Kbps.
  • The Live API accepts H264 encoded video and AAC encoded audio only.
  • The maximum length for a live stream is 90 minutes.

How to start a stream Using Open Broadcaster Software

  1. Download Open Broadcaster Software

    Open Broadcaster Software is free and open source software for video recording and live streaming.
    Download it from https://obsproject.com/

  2. In Open Broadcaster, Click on settings

  3. In settings, under encoding, make sure encoder is set to x264, Use CBR is enabled, and Max Bitrate is set to 2500

  4. Under Broadcast Settings, Set Streaming Service as Custom,
    the FMS URL as ***,
    and the Play Path/Sream Key as ---

  5. Under Video, make sure the resolution downscale is set to a value so that the maximum resolition is 1280x720, and the FPS is set to 30

  6. Under Advanced, make sure the Keyframe Interval is set to 1.

  7. Start streaming!
